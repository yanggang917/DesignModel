# DesignModel

一  设计模式的定义：
	设计模式就是在针对编码过程中遇到的问题总结出来的最佳解决方案。

二  解决的问题：
	要解决的问题就是代码可维护性问题、复用性问题、扩展性问题、灵活性问题。
